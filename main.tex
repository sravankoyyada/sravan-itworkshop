\section{About Iris Data}
The following data includes three species iris flower with 50 samples each as well as
some properties about each flower.
The columns in this dataset are:
\begin{itemize}
\color{brown}
\item Sepal Length
\color{red}
\item Sepal Width
\color{cyan}
\item Petal width
\color{lime}
\item Petal Width
\color{teal}
\item Species
\end{itemize}
